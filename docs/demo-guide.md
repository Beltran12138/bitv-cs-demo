# bitV 客服机器人 Demo 使用指南

## 访问地址

| 角色 | 链接 |
|------|------|
| 用户端（模拟客户） | https://cs-demo-beta.vercel.app/chat |
| 客服端（模拟坐席） | https://cs-demo-beta.vercel.app/agent |

---

## 演示流程（建议双屏或双标签页）

### 第一步：准备
用两个标签页分别打开**用户端**和**客服端**。

### 第二步：用户与机器人对话
在用户端点击右下角 **💬 在线客服** 按钮，输入以下任意关键词触发机器人回复：

| 输入示例 | 机器人回答主题 |
|----------|---------------|
| 手续费 / fee | 交易手续费说明 |
| 提币 / withdraw | 提币流程和时间 |
| KYC / 实名 | KYC 验证说明 |
| 充值 / deposit | 充值方式 |
| 安全 / security | 账户安全说明 |
| 合约 / futures | 合约交易说明 |

### 第三步：转接人工客服
两种方式触发转人工：
- 点击输入框上方的 **「转人工客服」** 按钮
- 连续发送 3 条机器人无法识别的消息（自动触发）

### 第四步：客服接入
切换到**客服端**，可看到该会话以黄色高亮出现在左侧列表，点击 **「接入」** 按钮。

### 第五步：实时对话
客服端右侧输入框发送消息，用户端立即收到（延迟 < 1 秒）。用户端界面无任何变化，实现无感知切换。

---

## 语言切换

用户端聊天窗口顶部可切换三种语言：**简体中文 / 繁體中文 / English**，机器人和界面文本同步切换。

---

## 注意事项

- 客服端无需登录，知道 URL 即可访问（Demo 阶段）
- 每次打开聊天窗口会创建一个新会话
- 页面刷新后历史记录不会恢复（Demo 阶段不做持久化登录）
