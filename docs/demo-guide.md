# bitV 客服机器人 Demo 使用指南

## 访问地址

| 角色 | 链接 |
|------|------|
| 用户端（模拟客户） | https://cs-demo-beta.vercel.app/chat |
| 客服端（模拟坐席） | https://cs-demo-beta.vercel.app/agent |

---

## 演示流程（建议双屏或双标签页）

### 第一步：准备
用两个标签页分别打开**用户端**和**客服端**。

### 第二步：与 AI 机器人对话
在用户端点击右下角 **💬 在线客服** 按钮，直接用自然语言提问，AI 会自动理解并回答：

| 提问示例 | 对应专家 |
|----------|---------|
| 手续费是多少？/ What are the fees? | 费率专家 |
| 我想提币，怎么操作？ | 提币专家 |
| KYC 认证需要什么材料？ | KYC 专家 |
| 如何充值？/ How do I deposit? | 充值专家 |
| 账户安全怎么保障？ | 安全专家 |
| 合约最高杠杆是多少？ | 合约专家 |
| 怎么注册账号？ | 注册专家 |
| 如何使用 API？ | API 专家 |

> AI 支持多轮对话，可以追问、补充细节，无需使用特定关键词。

### 第三步：转接人工客服
两种方式触发转人工：
- 点击输入框上方的 **「转人工服务」** 按钮
- 输入"人工"、"客服"、"转人工"等词（自动识别）
- 连续 3 次 AI 无法回答（自动触发）

### 第四步：客服接入
切换到**客服端**，可看到该会话以黄色高亮出现在左侧列表，点击 **「接入」** 按钮。

### 第五步：实时对话
客服端右侧输入框发送消息，用户端立即收到（延迟 < 1 秒）。用户端界面无任何变化，实现无感知切换。

---

## 语言切换

用户端聊天窗口顶部可切换三种语言：**简体中文 / 繁體中文 / English**，AI 回复语言同步切换。

---

## 安全机制

若用户消息中包含微信、Telegram、电话号码等引流词，系统自动回复官方渠道提示，不会转入人工流程。

---

## 注意事项

- 客服端无需登录，知道 URL 即可访问（Demo 阶段）
- AI 回复由 DeepSeek 驱动，首次回复约 1–3 秒，期间有打字动画
- 若等待人工超过 3 分钟，系统自动发送提醒消息
- 每次打开聊天窗口会创建一个新会话
